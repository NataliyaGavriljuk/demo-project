Система контролю версій  Git
-----------------------------

При виконанні цього завдання я зробила наступне:

1. Завантажила систему **Git** із сайту [http://git-scm.com](1)
2. Встановила початкові налаштування імені та електронної адреси:

```
$ git config --global user.name "Nataliya Gavriljuk"
$ git config --global user.email "ntk.gavriljuk@mail.ru"
```
 3.Згенерувала ***ssh-ключ***
 
 При роботі з ключем у мене виникли деякі проблеми, одна з яких:
 
  - не можна було додати ключ до агента. Появлялася помилка
  
```Could not open a connection to your authentication agent.```
 
Проблема була в тому, що агент не завантажувався автоматично. Вручну агент можна завантажити, використовуючи команду:

```eval $(ssh-agent -s) ```
